# Zoom
A full stack video conferencing web application built with react , node.js , Webrtc , Socket.io and Material ui . The app enables real time video/audio communication with signaling handled via socket.io and peer-to-peer streaming using webrtc.

#Tech Stack
-frontend
React 
Material UI
WebRTC
Socket.io client

-backend
Node.js
Express.js
Socket.io
Webrtc Server
